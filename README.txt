Datasets:
POGOH Station Locations October 2024 (https://data.wprdc.org/dataset/station-locations/resource/1498850a-86c9-4fa6-b4b1-9f4fd76abe19)
This dataset contains information about POGOH bikeshare stations, including their street locations, total dock counts, and geographic coordinates.

Monthly_Updating_Bus_Stop_Usage (https://data.wprdc.org/dataset/prt-transit-stop-usage/resource/3f40b94b-4ac4-48f1-8c61-8439d2d2f420?inner_span=True)
This dataset provides the average number of riders entering and exiting bus stops across the greater Pittsburgh region, along with the geographic coordinates of each stop.

City of Pittsburgh Crosswalks (https://data.wprdc.org/dataset/city-of-pittsburgh-markings/resource/632fbb91-c55d-4221-a8ad-91c72902bc61?inner_span=True)
This dataset contains information on crosswalks, including their geographic coordinates, associated neighborhoods, and crosswalk types.

Repository Overview: This project aims to identify the most desirable neighborhood in Pittsburgh based on access to public transportation infrastructure hence the neighborhoods with a higher number of bikeshare stations, bus stops, and crosswalks are considered more favorable for their accessibility and commuter convenience.
